# 🧠 Quiz de Python

Quiz de múltipla escolha sobre Python rodando no terminal.

## O que faz
- Carrega perguntas de um arquivo JSON
- Embaralha as perguntas a cada rodada
- Valida respostas e dá feedback instantâneo
- Salva histórico de pontuações com data e hora

## Como rodar
```bash
python quiz.py
```

## Tecnologias
- Python 3
- JSON
- Módulos: `json`, `random`, `datetime`
